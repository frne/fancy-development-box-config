## Notes

This config should work on every OS/distro with a bash console. The configuration file name is maybe ```~/.bashrc``` or ```~/.profile```.

Tested with:
- Debian Squeeze
- Debian Wheezy
- OSX 10.6
- OSX 10.7
- OSX 10.8
